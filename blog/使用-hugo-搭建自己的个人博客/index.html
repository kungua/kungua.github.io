使用 Hugo 搭建自己的个人博客

安装 Hugo

  1. 访问 Hugo官网
  2. 找到 Quick install 使用命令行执行下面几个命令 (开始之前请确定你安装过 brew 否则请自行百度如何在你的环境下安装 Hugo)
    1. brew install hugo

使用 Hugo 在本地生成博客站点

  1. 访问 Hugo官网的 Quick Start 部分, 依次执行下面几行命令
hugo new site `Your blog site`

(建议 Your blog site 可以是 GitHubUsername.github.io.generator)

cd `Your blog site`

git init

git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke

echo 'theme = "ananke"' >> config.toml

这一步 在安装和配置主题～

hugo new posts/`Your first post's title`.md

这一步会在项目目录的根目录下生成 /content/posts/xxxx.md 使用你喜欢的编辑器打开它, 里面默认的内容是元信息不要动, 在下面添加你的博客内容, 写完之后去把元信息的 draft 改成 false

hugo server -D

这一行开启本地预览,命令行会显示一个可以在浏览器预览的地址, 默认是 http://localhost:1313/

你可以在 config.toml 这个文件中配置你的主题～

hugo

这一行会生成 public 文件夹, 先把这个文件加入到 .gitignore 里面, 然后将这个文件夹推到远程你的叫 username.github.io 的仓库并配置 GitHub Page 就可以完成预览啦～

修改你的主题

访问 Hugo 主题网站 挑选你喜欢的主题, 但是使用不同主题可能会有一些不同的配置项， 本文只针对文章中出现的主题。

以 一只狗狗的pulp 主题为例

git submodule add https://github.com/koirand/pulp.git themes/pulp

然后将项目根目录下的 /theme/pulp/exampleSite 下面的内容复制到根目录下

将 /content/blog/ 目录下的一个 rich 打头的 md 文件删掉。然后在这个目录新建你的博客就好啦。 记得自己添加元信息。
